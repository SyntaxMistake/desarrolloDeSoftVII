## Descripción del Proyecto

Este repositorio contiene la solución a la **Práctica #1** de la asignatura **Desarrollo de Software VII**, correspondiente al I Semestre 2026.

Se implementan 5 ejercicios en PHP que demuestran:
- **Variables y tipos de datos** (strings, integers)
- **Operadores aritméticos** (+, -, *, /, %, **)
- **Orden de precedencia** de operadores
- **Estructuras repetitivas** (for loops)
- **Arreglos asociativos** y recorrido con foreach
- **Condicionales** (if statements)
